# LGTM Dashboards

## LGTM Production Stack Overview

Apply:

```bash
make dashboards
```

Open:

```text
Grafana > Dashboards > Stack Overview > LGTM Production Stack Overview
```

Dashboard nay the hien:

- Grafana
- Prometheus
- Alertmanager
- Mimir
- Loki
- Tempo
- Alloy Gateway
- Alloy Agent
- MinIO sandbox object storage
- PVC/storage
- Logs tu Loki

Thong diep demo:

```text
LGTM la stack modular. Dashboard nay cho thay toan bo cac backend dang chay, dung tai nguyen bao nhieu, co restart khong, storage co on khong, va logs gan nhat nam o dau.
```
