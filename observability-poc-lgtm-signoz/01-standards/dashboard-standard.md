# Dashboard Standard

## Folder structure đề xuất

```text
/Executive
/NOC
/Platform
/Kubernetes
/Applications
/Logs
/Traces
/SLO
/Incident Investigation
/Capacity
```

## Dashboard POC tối thiểu

1. Executive POC Dashboard
2. NOC POC Dashboard
3. Kubernetes Overview
4. Service Overview Template
5. Log Analysis
6. Incident Investigation

## Service Dashboard phải có

```text
- Request rate
- Error rate
- p50/p95/p99 latency
- HTTP status code
- CPU/memory
- Pod restart
- Recent error logs
- Trace examples
- Owner/team/criticality
```
